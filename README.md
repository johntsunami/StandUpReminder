# StandUp Reminder

A tiny, **dependency-free** desktop reminder that nudges you to **STAND UP** after
sitting too long, then to **SIT DOWN** after a shorter standing break. Built for
managing lumbar/sacral back pain by regularly taking the load off your spine.

When the timer fires, a **transparent, always-on-top popup** appears over every
other window with a big message (and a rotating humorous line to keep it fun).

## Why it runs anywhere

It uses **only the Python standard library** (`tkinter`, `winsound`, `json`,
`datetime`). There is **nothing to `pip install`** — so it works even on
locked-down corporate / Citrix / HIPAA machines that allow Python scripts but
block package installs.

## Requirements

- **Python 3.7+** (Windows ships fine; Tk is bundled with the standard installer).
- No other dependencies.

## Install & run

```bash
# 1. Get the files (clone or download the ZIP)
git clone https://github.com/<your-username>/StandUpReminder.git
cd StandUpReminder

# 2. Run it
python standup.py
```

On Windows you can also **double-click `run.bat`** (runs with no console window),
or just download `standup.py` and run `python standup.py`.

### Put an icon on your Desktop (per computer)

So you can always start it yourself (e.g. if auto-start is disabled on a managed
work machine), run this once on each computer:

```bash
python install.py
```

It drops a **"StandUp Reminder" icon on your Desktop** with a custom icon. It
tries three methods (native Windows COM, then PowerShell, then a plain `.bat`)
so it still works on locked-down machines. Double-click the Desktop icon anytime
to launch the app.

> The icon file `standup.ico` is generated by `make_icon.py` (pure standard
> library). It's included in the repo, but `install.py` will regenerate it if
> missing.

## Using it

- A small control window shows your current state and countdown.
  - **Pause / Start**, **Reset**, **Skip** (Skip fires the next popup now — useful for testing).
- **Settings** (menu): sit time (default 45 min), stand time (default 5 min),
  snooze length, sound on/off, popup transparency, keep-on-top, and
  **start automatically at login**.
- **Quotes** (menu): add, delete, **import** (`.txt`, one quote per line, or `.json`
  list), export, or restore the built-in defaults. Separate lists for STAND UP
  and SIT DOWN.
- On the popup itself:
  - **✅ I'm Standing/Sitting** — acknowledge and start the next phase.
  - **😴 Snooze** — remind me again shortly.
  - **👎 Delete quote** — never show that line again.
  - **🔄 New quote** — shuffle to a different line.
  - `Esc` also dismisses.

## Where your data lives

Settings and quotes are stored in your home folder so they survive updates and
work on read-only install locations:

```
~/.standup_reminder/config.json
~/.standup_reminder/stand_quotes.json
~/.standup_reminder/sit_quotes.json
```

Want more humor? Add lines in the Quote Manager, or just ask Claude to generate a
batch and **Import** them.

## Auto-start at login (Windows)

Enable it in **Settings → "Start automatically at login"**. This drops a small
`.bat` in your per-user Startup folder (no admin rights needed) that launches the
app with `pythonw` (no console window). Disable it the same way.

## Publishing / updating on GitHub

First time (create the repo on github.com as an **empty** repo -- no README,
no .gitignore, no license -- then):

```bash
cd StandUpReminder
git init
git branch -M main
git remote add origin https://github.com/<your-username>/StandUpReminder.git
git add .
git commit -m "StandUp Reminder"
git push -u origin main
```

Pushing later updates:

```bash
git add .
git commit -m "Describe what changed"
git push
```

On another computer, grab the latest version with:

```bash
git pull
```
